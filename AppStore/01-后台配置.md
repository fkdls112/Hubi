## 年龄分级问卷答案（全部选"无"）

| 项 | 频率 |
|---|---|
| 卡通或幻想暴力 | 无 |
| 现实暴力 | 无 |
| 性内容或裸露 | 无 |
| 不雅或粗俗幽默 | 无 |
| 成熟、暗示性主题 | 无 |
| 恐怖、惊悚主题 | 无 |
| 模拟赌博 | 无 |
| 医药/医疗信息 | 无 |
| 酒精、烟草或毒品引用 | 无 |
| 极少/中度仿真赌博 | 无 |
| 用户生成内容 | 无 |
| 不受限制的网络访问 | 是（用户配置 AI Provider 时） |

> "不受限制的网络访问 = 是"——因为用户可填自己的 OpenAI/Anthropic 端点。这会自动把分级提到 17+。
> 替代方案：在 App 内强制只允许预设端点（OpenAI/Anthropic 官方），但会牺牲灵活性。
> **推荐：保持 17+ 等级**，与 ChatGPT、Claude 官方 App 一致。
