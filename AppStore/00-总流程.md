## Phase 2: App Store Connect 后台准备

### 2.1 登录 https://appstoreconnect.apple.com（fkdls112 对应 Apple ID）

### 2.2 协议 / 税务 / 银行
- [ ] 同意 Paid Apps Agreement（"协议、税务和银行业务"）
- [ ] 填税务表（CN W-8BEN）
- [ ] 绑定大陆银行账户

> 这步不做就**没法卖收费 IAP**，必须先做。
> 银行卡要求：本人持卡，开户行不能是网络银行（如微众）。

### 2.3 创建 App（参考 `01-后台配置.md`）
- [ ] 创建新 App：iOS / Hub-i / com.hubi.ap / hubi-app-001
- [ ] 填副标题、隐私政策 URL、营销 URL、客户支持 URL
- [ ] 填年龄分级问卷
- [ ] 填类别：工具 + 生活

### 2.4 创建 IAP 产品（参考 `06-IAP配置.md`）
- [ ] 创建 1 个非消耗型产品 `com.hubi.lifetime`（¥49 终身买断）
- [ ] 创建 1 个非消耗型产品（lifetime）
- [ ] 每个产品填中英本地化
- [ ] 每个产品上传一张审核截图（Paywall）

> **注意**：IAP 必须单独审核，可与主 App **同步提交**。建议在主 App 第一次提交时一起提交。

## Phase 5: 审核中

| 状态 | 时长 | 动作 |
|---|---|---|
| Waiting for Review | 1-3 天 | 等 |
| In Review | 数小时 - 1 天 | 等 |
| Pending Developer Release | 已通过 | 你点"发布" |
| Rejected | - | 看 Resolution Center 改 |

**常见拒绝原因**：
- IAP 价格未填全 → 补全
- 隐私政策链接 404 → 检查
- 测试账号失效 → 重新提供
- 17+ 解释不清 → 在备注里强调
