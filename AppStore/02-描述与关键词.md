Hub-i 是一只可爱的橘色小老虎，也是你的 AI 聊天助手。

不必再为不同 AI 切换 App，一个 Hub-i 就能搞定。

• 多平台 AI 对话：OpenAI / Anthropic Claude / Google Gemini / 国产模型，一键切换
• 自带 Key 模式：带上你自己的 API Key，免费聊到天荒地老
• 智能 Agent：预置写作、翻译、编程、占星等多种角色，也可自定义
• 图片对话：拍照或选图，让 AI 帮你分析
• 本地优先：所有对话存在你的设备上，端到端备份到 iCloud

【高级版】
• 免费版：自填 Key 即可接入任意自定义模型，功能完整
• 终身高级版（¥49 一次付费）：永久解锁全部预设模板与预设智能体

【隐私至上】
• 你的对话只存本机，不上传任何服务器
• API Key 用 iOS Keychain 加密存储
• 备份文件可加密导出，密码只有你知道

【设计哲学】
极简、克制、可爱——
没有广告，没有臃肿的功能，没有让你"上瘾"的设计。
就是一个干净、好用、能陪你聊天的小工具。

下载 Hub-i，让小老虎陪你写作、思考、探索。
